# جذور فلسطين (Palestine Roots)

تطبيق "جذور فلسطين" هو منصة رقمية تهدف إلى توثيق وحماية التراث المعماري والتاريخي الفلسطيني. يتيح التطبيق للمستخدمين استكشاف المواقع الأثرية، المساجد، الكنائس، والبيوت القديمة عبر تجربة تفاعلية حديثة.

## المميزات الرئيسية
- **خريطة تفاعلية:** عرض المواقع الأثرية على خريطة جوجل مع إمكانية النقر للتفاصيل.
- **استكشاف متطور:** البحث والفلترة حسب المحافظات (10 محافظات) والتصنيفات (ديني، تاريخي، طبيعي، تراثي).
- **الوضع الليلي:** دعم كامل للثيم الداكن لحماية العين وتوفير الطاقة.
- **المفضلة:** إمكانية حفظ المواقع للوصول إليها لاحقاً.
- **مشاركة:** مشاركة معلومات المواقع مع الآخرين.
- **دعم اللغتين:** العربية والإنجليزية.

## المعمارية التقنية (Architecture)
يتبع المشروع نمط **Clean Architecture** لضمان قابلية الاختبار والصيانة:
- **Data Layer:** Room (قاعدة بيانات محلية) + DataStore (إعدادات) + Gson (تحليل JSON).
- **Domain Layer:** Model و Repository Interface.
- **Presentation Layer:** مبنية باستخدام XML Views مع ViewBinding ونمط MVVM.

## التقنيات المستخدمة
- **Kotlin 1.8.10** مع **ViewBinding** لبناء واجهات المستخدم.
- **Hilt (Dagger)** لحقن التبعيات.
- **Room Database** للتخزين المحلي مع KAPT.
- **Coroutines & Flow** لإدارة العمليات الخلفية وتدفق البيانات.
- **Glide** لتحميل الصور بكفاءة.
- **Google Maps SDK** لعرض الخرائط.
- **DataStore** لحفظ الإعدادات.
- **Material Components** للتصميم.

## متطلبات التشغيل
- **JDK:** 11 أو أحدث
- **Android Studio:** Chipmunk | 2021.2.1 أو أحدث
- **Gradle:** 7.4
- **AGP:** 7.2.2
- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 33

## التحديثات الأخيرة (إصلاحات النسخة النهائية)
- **ترقية البيئة:** تحديث AGP و Gradle و Kotlin لأحدث الإصدارات المستقرة المتوافقة.
- **إصلاح البيانات:** حل مشكلة اختفاء المواقع بسبب تعارض مسميات الحقول (Snake_Case vs CamelCase) باستخدام `@SerializedName`.
- **استقرار الواجهات:** تفعيل View Binding بشكل كامل لضمان عدم حدوث Crash عند التنقل.

## كيفية التشغيل
1. قم بعمل Clone للمستودع.
2. افتح المشروع باستخدام Android Studio.
3. تأكد من أن JDK 11 محدد في إعدادات المشروع.
4. قم بعمل Sync Gradle وانتظر اكتمال التحميل.
5. قم ببناء المشروع وتشغيله على جهاز حقيقي أو محاكي.

## بيانات المواقع
يحتوي التطبيق على 30 موقع أثري موزعة على 10 محافظات فلسطينية:
القدس، رام الله والبيرة، نابلس، جنين، طولكرم، قلقيلية، سلفيت، أريحا والأغوار، الخليل، بيت لحم.
